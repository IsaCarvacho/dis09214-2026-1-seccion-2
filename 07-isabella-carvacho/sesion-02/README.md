## Markdown  
Markdown es un lenguaje de marcado ligero. Es una manera de escribir un texto con un formato, usando caracteres comunes del teclado.  

### Tipos de Markdown  
__Saltos de línea:__
1. Para separar dos parrafos se __presiona Enter dos veces__.
2. Para hacer que un texto baje una línea sin dejar un espacio en blanco, se debe poner __dos espacios en blanco (espacio) al final de la frase y luego Enter__
3. Forzar el salto es con <br> 
* Sangría con TAB o 4 espacios.
* --- Para separar secciones con una línea.

__Encabezados:__  
Para los encabezados se usan los #, dejando un espacio entre hashtag y encabezado. Para el título más grande se usa un hashtag al pricipio, espacio y luego la palabra. Se van sumando hashtags para hacer que el encabezado reduzca su tamaño.

__Formatos de texto:__  
__Negrita__ Dos astericos al inicio y final o asteriscos

_Cursiva_ Un guión bajo al inicio y final o asterisco 

***Negritaycursiva*** Tres asteriscos al inicio y final

~~Tachado~~ Dos ~~ al principio y al final

__Para listas__

***Viñetas***
* Elemento A
* Elemento B
    * Sub elementoC1
--> Se pone un *, espacio y luego el elemento a escribir. Dos espacios y enter para que se introduzca el siguiente elemento automáticamente. 

***Números***
1. Elemento A
2. Elemento B
    1. Sub elemento C1
--> Se pone un número, seguido de un punto, espacio y luego el elemento a escribir. Dos espacios y enter para que se introduzca el siguiente elemento automáticamente. 

***Checkbox***  
-[ ] Elemento X  
-[x] Elemento Y  
-[ ] Elemento Z
--> Se pone un [ seguido de un espacio o una x según corresponda,seguido de el ] para cerrar, espacio y luego el elemento a escribir.
